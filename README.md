# horarios-ucc

## Descripción
Sistema para generar horarios académicos automáticamente, gestionando docentes, aulas, clases y períodos.

## Funcionalidades

- Cargar clases desde CSV
- Registrar docentes
- Registrar aulas
- Configurar períodos
- Generar horarios automáticamente
- Asignación manual

## Módulos

### Configuración
- Año
- Período
- Turno (Sabatino, Diario)

### Docentes
- Nombre
- Especialidad

### Aulas
- Tipo (Aula, Laboratorio, Taller)

### Clases
- Nombre
- Créditos
- Tipo

### Horarios
- Generación automática
- Asignación manual
